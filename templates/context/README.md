# Context Folder

Drop any reference files here. All Chief Clarity agents read these as **read-only** context — they never modify files in this folder.

## What to put here

Any file that helps Chief Clarity give you better advice:

- **Expenses** — monthly budget, spending log, bank export
- **Health** — workout log, sleep tracker export, meal plan
- **Work** — meeting notes, project briefs, job descriptions
- **Finance** — investment summary, tax documents, loan details
- **Family** — school schedules, activity calendars, contact lists
- **Reference** — research notes, reading lists, bookmarks

## Format

Any format works — `.md`, `.txt`, `.csv`, `.json`. As long as the AI can read it, Chief Clarity can use it.

## Examples

- `expenses.md` — monthly spending by category
- `workout_log.md` — exercise sessions and notes
- `meeting_notes.md` — recent meeting summaries
- `kids_schedule.md` — school and activity calendar

## Rules

- Chief Clarity agents **never modify** files in this folder
- You own and maintain these files yourself
- More context = better suggestions, better pattern detection, better answers
