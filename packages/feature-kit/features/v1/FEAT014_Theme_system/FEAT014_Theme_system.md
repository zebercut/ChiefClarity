# F14 — Theme system

Dark and light theme with persisted user preference.

---

## What this delivers

Users can switch between dark and light themes. The choice persists across sessions and applies instantly.

## Capabilities (shipped)

- **Toggle** — button in the app header switches between dark and light mode.
- **Persistence** — preference saved to local storage, restored on app launch.
- **System-aware** — defaults to the OS preference on first launch.
