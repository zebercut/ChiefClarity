# F01 — Task management dashboard

Browse, search, filter, and edit all tasks from a dedicated view with batch operations and multiple layout options.

---

## What this delivers

A single place to see and act on every task — not just what the LLM chose to surface in today's plan. Users can find overdue items, bulk-resolve stale tasks, and edit priorities without going through the chat.

## Key capabilities

- **Searchable task list** — tap the "N tasks" chip on the Focus Dashboard to open a full list. Filter by status, priority, due date, or tag.
- **Overdue tracking** — tap the "N overdue" chip for a filtered view showing only past-due items. Batch actions: snooze all, reschedule, or mark done.
- **Inline editing** — change priority, due date, status, or notes directly in the list without round-tripping through chat.
- **Kanban layout** (optional) — toggle between list and kanban columns (To Do / In Progress / Done) with drag-and-drop reordering.

## User stories

- As a user, I want to see all my tasks at a glance so I can spot things the daily plan missed.
- As a user, I want to bulk-snooze overdue tasks on a busy day without chatting about each one.
- As a user, I want to change a task's priority without leaving the task view.

## Out of scope

- Task creation (handled via chat)
- Subtasks or task dependencies
- Shared/collaborative task lists
