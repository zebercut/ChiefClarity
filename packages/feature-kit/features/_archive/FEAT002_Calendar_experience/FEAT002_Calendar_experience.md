# F02 — Calendar experience

Full calendar with day, week, and month views, time-blocked agenda, and inline event editing.

---

## What this delivers

A visual calendar that lets users see beyond today's agenda. Navigate forward/backward, spot scheduling gaps, and edit events directly — no need to describe changes in chat.

## Key capabilities

- **Today's agenda** — tap the "Next: event" chip on the Focus Dashboard for a time-blocked view of today, showing free/busy slots and event details.
- **Week and month views** — swipe or tap to switch between day, week, and month. Tap any day to drill into its agenda.
- **Inline event editing** — tap an event to reschedule (drag to new time), cancel, or add notes. Changes write directly to calendar.json.
- **Visual density** — color-coded by category (work, personal, family). Conflicts highlighted with a warning badge.

## User stories

- As a user, I want to see my entire week at a glance to plan ahead.
- As a user, I want to drag an event to a new time without typing a chat message.
- As a user, I want to spot calendar conflicts visually before they become problems.

## Out of scope

- Event creation (handled via chat or inbox)
- External calendar write-back (read-only sync is F06)
- Shared calendars
